Modal Component

A customizable modal dialog component that can be easily integrated into your web applications.

Live Demo

Demo Modal

This is a demo of the modal component.

You can customize its appearance using the controls on the right.

Attributes

Name Type Default Description
--modal-width CSS length 300px Sets the width of the modal
--modal-height CSS length auto Sets the height of the modal
--modal-background-color color #ffffff Sets the background color of the modal

Events

Event Name Description Detail
modal-opened Fired when the modal is opened None
modal-closed Fired when the modal is closed None

Methods

Name Parameters Description
open() None Opens the modal
close() None Closes the modal

Code Example

<button class="open-modal">Open Modal</button>

<custom-modal>
  <h3>Demo Modal</h3>
  <p>Modal content goes here</p>
</custom-modal>

<script>
  const modal = document.querySelector('custom-modal');
  const openButton = document.querySelector('.open-modal');
  openButton.addEventListener('click', () => modal.open());
</script>
