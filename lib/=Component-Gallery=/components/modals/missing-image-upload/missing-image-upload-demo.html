Missing Image Upload

A modal interface for uploading missing recipe images with preview functionality and Firebase integration. Features RTL support and automatic image processing.

Live Demo

Methods

Name Parameters Description
openModalForRecipe() recipeId: string Opens upload modal for specific recipe

Code Example

const uploader = document.querySelector('missing-image-upload');
uploader.openModalForRecipe('recipe-123');
