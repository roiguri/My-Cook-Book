Recipe Filter Component

A modal interface component for filtering recipes with multiple criteria. Features RTL support, real-time updates, and integrates with Firebase/Firestore for data management.

Live Demo

Attributes

Name Type Default Description
category string null Optional category to filter recipes by
cooking-time-filter boolean true Enable/disable cooking time filter
difficulty-filter boolean true Enable/disable difficulty filter
ingredient-filter boolean true Enable/disable main ingredient filter
tags-filter boolean true Enable/disable tags filter
favorites-only boolean false When true, only shows user's favorite recipes

Events

Event Name Description Detail
filter-applied Fired when filters are applied { recipes: Array, filters: Object }
filter-reset Fired when filters are reset { category: string }
modal-opened Inherited: Fired when modal opens None
modal-closed Inherited: Fired when modal closes None

Methods

Name Parameters Description
open() None Opens the filter modal
close() None Closes the filter modal
clearFilters() None Resets all filters to default state
