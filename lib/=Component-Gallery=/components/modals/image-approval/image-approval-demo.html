Image Approval Modal

A modal interface for approving or rejecting recipe images with Firebase integration. Features RTL support and image preview functionality.

Live Demo

Events

Event Name Description Detail
image-approved Fired when image is approved { recipeId: string }
image-rejected Fired when image is rejected { recipeId: string }
modal-opened Inherited: Fired when modal opens None
modal-closed Inherited: Fired when modal closes None

Methods

Name Parameters Description
openModalForImage() imageData: Object Opens modal with specified image data
closeModal() None Closes the modal and resets state

Code Example

const approvalModal = document.querySelector('image-approval-component');

// Open modal with image data
approvalModal.openModalForImage({
    imageUrl: 'path/to/image.jpg',
    recipeId: 'recipe123',
    recipeName: 'Chocolate Cake'
});

// Handle approval/rejection events
approvalModal.addEventListener('image-approved', (e) => {
    console.log('Image approved for recipe:', e.detail.recipeId);
});

approvalModal.addEventListener('image-rejected', (e) => {
    console.log('Image rejected for recipe:', e.detail.recipeId);
});
