Confirmation Modal Component

A customizable confirmation dialog component that extends the base modal with approve/reject actions. Supports RTL layout and customizable messages.

Live Demo

Attributes

Name Type Default Description
--primary-color color #bb6016 Primary color for approve button
--primary-hover color #5c4033 Hover color for buttons
--secondary-color color #e6dfd1 Background color for reject button

Events

Event Name Description Detail
confirm-approved Fired when the approve button is clicked None
confirm-rejected Fired when the reject button is clicked None
modal-opened Inherited: Fired when modal opens None
modal-closed Inherited: Fired when modal closes None

Code Example

const modal = document.querySelector('confirmation-modal');

// Default Hebrew confirmation
modal.confirm();

// Custom English confirmation
modal.confirm(
    'Delete this item?',
    'Confirm Delete',
    'Delete',
    'Cancel'
);

// Handle responses
modal.addEventListener('confirm-approved', () => {
    console.log('Action approved!');
});

modal.addEventListener('confirm-rejected', () => {
    console.log('Action rejected!');
});
